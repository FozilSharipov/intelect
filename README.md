# INTELLECT Education Center

## Project Overview
Create a modern, multilingual (Russian/Tajik), fully responsive website for the **INTELLECT Education Center** — a leading educational center in Dushanbe (Tajikistan) specializing in preparing applicants for centralized testing (NCT) and foreign language studies.

## Technical Requirements
- HTML5
- Tailwind CSS 3.4+
- Vanilla JS + Alpine.js
- GSAP 3.12
- EmailJS
- Yandex Maps
- Swiper.js 11+
- i18next
