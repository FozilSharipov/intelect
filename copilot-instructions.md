# Project Setup Instructions

## Extensions to Install
- GitHub Copilot
- Tailwind CSS IntelliSense
- Prettier
- Live Server
- Auto Rename Tag
- Bracket Pair Colorizer

## Development Steps
1. **HTML Structure**: Create the basic HTML files as per the structure.
2. **Tailwind CSS**: Configure Tailwind CSS using CDN.
3. **JavaScript Functionality**: Implement the required JS functionalities.
4. **Localization**: Set up i18next for language switching.
5. **SEO Optimization**: Add meta tags and schema.org markup.
6. **Testing**: Ensure responsiveness and performance metrics are met.
7. **Deployment**: Deploy the site using Vercel or GitHub Pages.

## Additional Notes
- Ensure to follow best practices for accessibility.
- Regularly commit changes to the repository.
- Use version control for managing updates.
- Ensure to follow the design system for colors and typography.
- Use the provided prompts for GitHub Copilot to generate code efficiently.
- Regularly check Lighthouse scores for performance and accessibility.
